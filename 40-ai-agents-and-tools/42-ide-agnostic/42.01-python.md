# 🐍 Python Development Tools

## Linting & Formatting
- **Ruff** – Fast all-in-one linter (replaces flake8, isort, pycodestyle, etc.)
- **Black** – Opinionated code formatter (use with Ruff for formatting consistency)

## Type Checking
- **mypy** – Static type checker for Python; battle-tested, widely used
- **pyright** – Faster alternative to mypy; excellent VS Code integration via Pylance

## Testing
- **pytest** – Simple, powerful testing framework (default choice)
- **pytest-cov** – Coverage reporting for pytest
- **pytest-mock** – Clean mocking utilities for pytest

## Dependency Management & Virtual Environments

### uv 
- Extremely fast package manager and virtual environment tool
- Modern replacement for pip + venv + pip-tools
- Use when: starting a new project or you want speed and simplicity|

## Configuration Management
- **pydantic-settings** – Type-safe settings and environment variable parsing (built on Pydantic)
- **python-dotenv** – Simple `.env` file loading (lighter alternative, no type safety)

## Code Quality & Git Hooks
- **pre-commit** – Run linters, formatters, and checks automatically before each commit

  Recommended `.pre-commit-config.yaml` hooks:
  ```yaml
  repos:
    - repo: https://github.com/astral-sh/ruff-pre-commit
      hooks:
        - id: ruff
        - id: ruff-format
    - repo: https://github.com/pre-commit/mirrors-mypy
      hooks:
        - id: mypy
  ```

## Debugging & Development
- **pdb / ipdb** – Built-in and improved interactive debuggers
- **rich** – Beautiful terminal output, better tracebacks, and structured logging
