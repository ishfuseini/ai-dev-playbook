# 🟨 JavaScript / TypeScript Development Tools

### Linting & Formatting
- **Biome** – All-in-one formatter + linter (modern replacement for ESLint + Prettier)

### Type Checking
- **TypeScript (tsc)** – Standard type checker
- **tsx** – Run TypeScript files directly without compiling (`tsx src/index.ts`)
- **ts-node** – TypeScript execution environment (more mature, more config; prefer `tsx` for new projects)

### Testing
- **Vitest** – Fast, modern test runner (Vite-native, Jest-compatible API)
- **Jest** – Widely used, more mature ecosystem (better choice if not using Vite)
- **Playwright** – End-to-end browser testing
- **Testing Library** – UI component testing utilities (framework-agnostic)

### Build Tools / Bundlers
- **Vite** – Fast dev server + build tool (default modern choice)
- **esbuild** – Ultra-fast bundler (often used under the hood by Vite and others)
- **tsup** – Simple TypeScript library bundler (built on esbuild)
- **Rollup** – Fine-grained bundler (good for libraries)

### Package Management
- **pnpm** – Fast, disk-efficient package manager (recommended for monorepos)
- **npm** – Default, widely supported
- **Bun** – Ultra-fast runtime + package manager (experimental for production use)

### Runtime & Execution
- **Node.js** – Standard runtime
- **Bun** – Fast all-in-one runtime (TypeScript-native, drop-in Node alternative)
- **Deno** – Secure-by-default runtime with built-in TypeScript support

### Dev Environment & DX
- **dotenv** – Environment variable management
- **zod** – Runtime schema validation + TypeScript type inference
