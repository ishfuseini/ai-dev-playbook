# MCP (Model Context Protocol) Servers

A reference list of MCP servers useful in AI-assisted development workflows.

> **Quality Note:** MCP server quality varies significantly. Where possible, prefer official or well-maintained servers.
> - ✅ **Official / Well-maintained** — published by the tool's creator or a trusted org, actively maintained
> - 🟡 **Community** — open source, useful, but verify activity and trust before use in production workflows

---

### 📁 Filesystem & Workspace
- **Filesystem MCP** ✅ – Read/write local files, traverse directories
- **Git MCP** ✅ – Repo history, diffs, branches, commits

---

### 🧠 Code Intelligence
- **Language Server MCP (LSP bridge)** 🟡 – Type info, definitions, references

---

### 🌐 Web & External Data
- **Browser MCP (Playwright/Puppeteer)** ✅ – Navigate, scrape, test UIs
- **Search MCP** 🟡 – Web search + retrieval

---

### 🧠 Context & Retrieval
- **Context7 MCP** ✅ – Up-to-date library documentation and semantic code search
- **Embedding / Vector DB MCP** 🟡 – Custom knowledge retrieval (docs, notes, internal wikis)

---

## Tips

- **Start minimal.** Only connect MCPs your agent actually needs for the task — fewer tools reduces hallucination risk and attack surface.
- **Audit community MCPs** before use: check last commit date, open issues, and whether the server requests excessive permissions.
- **Combine thoughtfully.** A Filesystem + Git + Postgres MCP covers most backend development tasks without adding noise.
