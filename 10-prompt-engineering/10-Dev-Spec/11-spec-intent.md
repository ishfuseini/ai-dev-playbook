# Prompt: Generate / Review a Project Intent Spec

Use this prompt in one of two ways:

**Mode A – Generate:** Paste a rough description of your project below and ask the AI to fill in the spec template.
> "Here is a rough description of my project: [your description]. Please fill in the Project Intent spec below as completely as possible, and flag any sections where you need more information."

**Mode B – Review:** Fill in the template yourself, then ask the AI to review it.
> "Here is my Project Intent spec. Please review it for gaps, ambiguities, unstated assumptions, and missing success criteria. Be direct — flag anything that could cause problems later."

---
