# Planning Tools

A curated reference of external tools for planning, spec creation, task management, and memory in AI-assisted development workflows.

---

## Task Management & AI Agents

Tools for breaking down and executing development tasks, including AI-driven task runners.

| Tool | Link | When to Use |
|------|------|-------------|
| **Hamster** | https://tryhamster.com | AI-assisted task tracking designed for developers — lightweight and fast |
| **Claude Task Master** | https://github.com/eyaltoledano/claude-task-master | Generate and manage a full task list from a PRD using Claude — great for solo AI-assisted projects |
| **Plandex** | https://github.com/plandex-ai/plandex | Autonomous AI coding agent for large, multi-step tasks across many files — better for execution than planning |

---

## Memory Systems

Tools for giving AI agents persistent memory across sessions.

| Tool | Link | Purpose |
|------|------|---------|
| **SpecStory** | https://specstory.com | Automatically captures and stores AI conversation history as searchable context |
| **Cursor Memory Bank** | https://github.com/vanzan01/cursor-memory-bank | Structured markdown-based memory system for Cursor — keeps AI context across sessions |
| **Mem0** | https://mem0.ai | Persistent memory layer for AI agents — stores user and project context across runs |

> **No memory tool?** See `30-project-setup/31-repo/31.01-documentation` for the manual Memory system using structured markdown files.
