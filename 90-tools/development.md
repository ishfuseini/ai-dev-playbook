# Development Tools

A curated reference of external tools that complement AI-assisted software development workflows.

## Code Intelligence & Navigation

| Tool | Link | Purpose |
|------|------|---------|
| **GitHub Copilot** | https://github.com/features/copilot | AI code completion and chat inside your IDE |
| **Cursor** | https://cursor.sh | AI-native code editor (VS Code fork) |
| **Zed** | https://zed.dev | Fast, collaborative editor with built-in AI |
| **Sourcegraph** | https://sourcegraph.com | Code search and navigation at scale |
